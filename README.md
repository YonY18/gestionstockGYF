# ** Gestor Stock y Presupuesto **

## Como Empezar
Primero: renombrar los .env.Ejemplo en .env
Segundo: abrir una consola en el backend y otra en el frontend
Tercero: iniciar los servidores con

```bash
npm run dev
```
